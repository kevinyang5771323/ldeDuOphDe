# 项目简介 python237

本项目是一个基于Python开发的联通用户管理系统，集成了登录注册、部门管理、用户管理、靓号管理、任务管理、订单管理和数据统计可视化等功能模块。

## 技术栈

### 后端

- Python
- Django框架

### 前端

- HTML
- CSS
- JavaScript

### 开发工具

- PyCharm

### 数据库

- MySQL

## 功能模块

- 登录注册
- 部门管理
- 用户管理
- 靓号管理
- 任务管理
- 订单管理
- 数据统计可视化

## 系统角色

系统包含以下角色：
- 用户
- 管理员

## 运行环境

- Python版本：3.7 - 3.8

1. 准备Python环境（3.7 - 3.8版本）
2. 安装依赖库（Django、MySQL等）
3. 配置数据库
4. 运行系统
